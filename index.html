Combinaison Linéaire

التركيب الخطي
التعريف

أي تعبير من الشكل:

u = λ₁v₁ + λ₂v₂ + ··· + λₙvₙ
✓

هو تركيب خطي: حللت النظام ووجدت قيم λᵢ بدون تناقض

✗

ليس تركيباً خطياً: ظهر تناقض في الحساب مثل 0 = 5

📝 مثال — هل u = (3, 1, 2) تركيب خطي من v₁=(1,0,1) و v₂=(0,1,0)؟

نحل: λ₁(1,0,1) + λ₂(0,1,0) = (3,1,2)

λ₁ = 3 | λ₂ = 1 | λ₁ = 2 ← تناقض! (3 ≠ 2)
✗

المعادلة الثالثة تعطي λ₁=2 لكن الأولى تعطي λ₁=3 ← ليس تركيباً خطياً

أما u = (3, 1, 3)؟  ← λ₁=3, λ₂=1, و المعادلة الثالثة: λ₁=3 ✓  ← هو تركيب خطي!

Famille Libre / Liée

مجموعة حرة / تابعة
طريقة الحل

حل المعادلة التالية وانظر في النتيجة:

λ₁v₁ + λ₂v₂ + ··· + λₙvₙ = 0
النتيجةالحكمالمعنى
λ₁=λ₂=···=0 فقط Libre ✓ لا يوجد متجه مبني من الباقين
وجد حل غير صفري Liée ✗ متجه واحد على الأقل = تركيب من الباقين
🔑 إذا عدد المتجهات أكبر من dim الفضاء ← المجموعة تابعة دائماً
📝 مثال — هل {v₁=(1,2,1), v₂=(0,1,2), v₃=(2,3,0)} libre في ℝ³؟

نحل: λ₁v₁ + λ₂v₂ + λ₃v₃ = (0,0,0)

λ₁ + 2λ₃ = 0
2λ₁ + λ₂ + 3λ₃ = 0
λ₁ + 2λ₂ = 0
①

من المعادلة (1): λ₁ = −2λ₃  |  من (3): λ₂ = −λ₁/2 = λ₃

②

نعوّض في (2): 2(−2λ₃) + λ₃ + 3λ₃ = −4λ₃ + 4λ₃ = 0 ← 0 = 0 (مجهول حر!)

✗

خذ λ₃=1 ← λ₁=−2, λ₂=1 ← وجد حل غير صفري ← المجموعة Liée

Famille Génératrice

مجموعة مولّدة
طريقة الحل

حل المعادلة لأي متجه عام في الفضاء:

λ₁v₁ + λ₂v₂ + ··· = (x, y, z)
النتيجةالحكم
دائماً توجد قيم λᵢ لأي x,y,zGénératrice ✓
في حالات لا يوجد حل (تناقض)Pas Génératrice ✗
📝 مثال — هل {v₁=(1,0,1), v₂=(0,1,1), v₃=(1,1,0)} génératrice لـ ℝ³؟

نحل: λ₁v₁ + λ₂v₂ + λ₃v₃ = (x, y, z)

λ₁ + λ₃ = x
λ₂ + λ₃ = y
λ₁ + λ₂ = z
①

نجمع المعادلات الثلاث: 2(λ₁+λ₂+λ₃) = x+y+z ← λ₁+λ₂+λ₃ = (x+y+z)/2

②

λ₁ = (x+y+z)/2 − y  |  λ₂ = (x+y+z)/2 − x  |  λ₃ = (x+y+z)/2 − z

✓

لأي (x,y,z) نجد λᵢ دائماً ← المجموعة Génératrice لـ ℝ³

Sous-espace Vectoriel

الفضاء الجزئي
الشروط الثلاثة

F ⊆ E فضاء جزئي إذا تحققت:

1

0 ∈ F — المتجه الصفري موجود في F

2

u,v ∈ F ⟹ u+v ∈ F — الجمع يبقى داخل F

3

u ∈ F, λ ∈ ℝ ⟹ λu ∈ F — الضرب بعددية يبقى داخل F

طريقة Vect — الأسرع والأذكى 🎯
1

اكتب المتجه العام بدلالة الشرط

2

فكّك بدلالة المتغيرات الحرة فقط

3

اكتب F = Vect{v₁, v₂, ...}

4

Base = المتجهات داخل Vect  |  dim = عدد المتغيرات الحرة

الشرط في ℝ³المتغيرات الحرةdimBase

y = 2x + z ← (x, 2x+z, z) = x(1,2,0) + z(0,1,1)

Intersection

التقاطع F ∩ G
المفهوم

المتجهات الموجودة في F و G معاً — وهو دائماً sous-espace.

F ∩ G = { u ∈ E : u ∈ F و u ∈ G }
1

اجمع كل شروط F وشروط G في نظام واحد

2

طبّق طريقة Vect كالعادة

🔑 كلما زادت الشروط ← dim أصغر
📝 مثال — أوجد F ∩ G مع Base و dim

F = {(x,y,z) ∈ ℝ³ : x + y = 0}    G = {(x,y,z) ∈ ℝ³ : y − z = 0}

الخطوة 1: نجمع الشروط:

x + y = 0 ← x = −y
y − z = 0 ← z = y
①

المتجه العام: (x,y,z) = (−y, y, y) = y(−1, 1, 1) ← المتغير الحر الوحيد هو y

✓

F ∩ G = Vect{(−1,1,1)}  |  dim(F∩G) = 1  |  Base = {(−1,1,1)}

Sous-espaces Supplémentaires

الفضاءان المتمّمان
التعريف

F و G متمّمان في E إذا تحقق شرطان معاً:

F + G = E و F ∩ G = {0}

ونكتب: E = F ⊕ G

الشرطالمعنى
F + G = Eمعاً يملآن كل الفضاء ✅
F ∩ G = {0}لا يتشاركان إلا الصفر ✅
🔑 طريقة التحقق السريعة:
E = F ⊕ G ⟺ dim(F) + dim(G) = dim(E) و F ∩ G = {0}
📝 مثال — هل F و G متمّمان في ℝ³؟

F = {(x,y,z) : x+y = 0 و z = 0}    G = {(x,y,z) : x = 0 و y = 0}

①

dim(F): F = Vect{(1,−1,0)} ← dim(F) = 1

②

dim(G): G = Vect{(0,0,1)} ← dim(G) = 1  |  dim(F)+dim(G) = 2 ≠ 3 ← ليسا متمّمين!

لو أخذنا بدلاً من ذلك: G' = Vect{(0,0,1),(1,0,0)} ← dim(G')=2

dim(F) + dim(G') = 1 + 2 = 3 = dim(ℝ³) و F ∩ G' = {0}
✓

الشرطان محققان ← ℝ³ = F ⊕ G' ← F و G' متمّمان!
